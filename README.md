# Actually Reliable Legal Policies

Canonical source for all [Actually Reliable](https://actuallyreliable.com) legal policies, published at [actuallyreliable.com/legal](https://actuallyreliable.com/legal) and applicable to all products and services.

## About Actually Reliable

Actually Reliable LLC is an Ohio-incorporated company providing software solutions. Our core values are **simplicity** and **security** — building straightforward, reliable products backed by clear playbooks and protocols.

## This Repository

Policies are maintained under a single legal framework covering all Actually Reliable products and one LLC entity, ensuring consistency for both us and our customers. They're written in Markdown with HUGO frontmatter for direct website integration via Git submodule.

**Data processing** is US-based by default, with limited exceptions when no US alternative exists or redundancy requires geographic distribution.
